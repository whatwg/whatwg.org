> > We have a premium video content available via subscription. We are

> > how copy protected we can be using the HTML5 video formats. At the
> > moment we are using the RTMP Streaming solution along with Flash DRM

>
> There is indeed already copy protection available for video content played
> using the HTML <video> element. Assuming you are based in the US, you can
> find more information about the copy protection available to this content
> at the following Web site:
>
>   http://www.copyright.gov/title17/
>
> This copy protection scheme also works for the HTML page, the CSS files,
> any JavaScript, and any images you may use as part of the video playback
> experience. Similar solutions exist in other countries as well.
