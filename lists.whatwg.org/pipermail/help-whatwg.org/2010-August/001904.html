Hi,

We are organizing an OpenClass on HTML 5 at New Delhi, India. Interested
people can register or know more at http://openclass.eventbrite.com/ .

The following are the topics which we will cover :

   - HTML 5
   - HTML 5 Canvas
   - HTML 5 Video
   - HTML 5 Audio
   - HTML 5 Local Storage
   - HTML 5 Geo Location
   - HTML 5 Appcache
   - HTML 5 New Tags
   - SVG and HTML 5
   - Role of WHATWG and W3C
   - Will HTML 5 mean the demise of Flash ?
   - HTML 5 Ecosystem - Browsers, Libraries etc
   - HTML 5 Standardization Process

Since it is an open class, we are ready to cover any topic which you want to
be covered in the class if it is related to HTML 5.
