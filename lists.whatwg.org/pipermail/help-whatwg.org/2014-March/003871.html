> Using floats to represent layout is actually not what floats are intended for

Just on that point, CSS can really be used however one pleases (as
long as separation of concerns is maintained, so, for example,
generated content should not be used to publish essays). That a CSS
feature has been designed with a purpose in mind doesn’t mean it’s
limited to that use.

The topic should probably go on a CSS mailing list though—a popular
one still seems to be http://www.css-discuss.org/ (where a similar
question got sent by another person it seems), and for spec matters
http://lists.w3.org/Archives/Public/www-style/.
