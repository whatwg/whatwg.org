If you're looking to automate the transcoding of a larger number of videos,
I use ffmpeg <http://www.ffmpeg.org/> and
ffmpeg2theora<http://v2v.cc/~j/ffmpeg2theora/>.
 They're both open source command line utilities that are very robust in the
formats they take as input.

Hope that helps,
