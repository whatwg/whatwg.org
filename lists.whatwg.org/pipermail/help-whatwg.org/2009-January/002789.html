Jason Cole wrote:
> I believe that <script> needs  type="text/javascript" to work. It certainly wouldn't hurt.

Unlike previous versions of HTML, HTML 5 has made the type attribute on 
script and style elements optional, and it makes little difference in 
browsers if it's included or not.  The spec defines the default to be 
text/javascript and that is how browsers behave.
