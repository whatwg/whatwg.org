I think we should punt on this for now, since the browser vendors have 
been so slow at implementing Web Forms 2 -- no point getting so far ahead 
of them that they can't even fathom doing everything the spec says.

I expect what we'll eventually do is introduce an API to do context- 
sensitive styling of textarea contents, so you can do things like source 
code syntax highlighting. This would also take care of your use case.
