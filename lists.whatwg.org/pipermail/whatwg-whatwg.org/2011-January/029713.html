I've scrolled the admittedly large message subject offscreen, the only
place the Subject is visible is in the window title, which I'm clearly
not reading. I'd only read that if I needed to find my way back to the

Subject to get back. Since this is OS X, if I want to get back, i'd

supported Aero Glass (it doesn't *sob*) or if someone else's Vista
computer did.
