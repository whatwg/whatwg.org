You're quite correct.  If you receive a response in this vein, your
best recourse is to try persuading at least one browser implementer to
ask for such a feature.  If browsers specifically want to implement it
now, Ian will probably be willing to spec it, if it's simple (clearly
we don't want unspecified features).  Otherwise, as he said, he'll
probably wait until browsers have implemented existing specs first.
Sometimes features get added without implementers' support, but don't
rely on it.  If it gets implemented, on the other hand, you can be
sure someone will want to spec it, so that interoperability doesn't
suffer.
