I see it in this terms: when I'm browsing the web I'm interested in docs
(well, maybe I'm not the most typical subject): no pop ups, zero useless
features, just reading. So, for a web surfer, 'javascript=off' is an
option.

**But**, if I'm interested in using a web app, there is no choice but to
activate the only language installed in my browser. To pretend use a web
app without javascript today is just as to try to see web pages without
a browser. 

'javascript=off' shouldn't be a concern in this list.
