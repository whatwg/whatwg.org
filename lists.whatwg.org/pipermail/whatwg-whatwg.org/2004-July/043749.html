Proposal: enhancing tabindex to allow focusability of custom HTML widgets:
http://www.mozilla.org/projects/ui/accessibility/tabindex.html

This is already supported in IE and Mozilla 1.8a2.

I would like to see Opera and Safari support as well. IBM is behind this 
as part of a larger effort to make DHTML accessible. There will always 
be a need to make custom widgets with Javascript, so there need to be 
mechanisms for making those widgets accessible.
