Note that it isn't because I don't like it (although I don't), it's
because the text/html RFC doesn't allow it. (As for as XML languages go,
only the profile of XHTML defined in XHTML 1 Appendix C may be sent as
text/html. And WF2 doesn't fall under that umbrella.)
