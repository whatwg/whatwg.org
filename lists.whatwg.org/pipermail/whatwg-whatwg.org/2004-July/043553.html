fantasai wrote:
> Several options:
> 
>   "-//WHATWG//DTD HTML4 + WF2//EN"
>   "-//WHATWG//DTD HTML4 + Web Forms 2.0//EN"
>   "-//WHATWG//DTD HTML4 with Web Forms 2.0//EN"
>   other

I suggested the following last month:

     "-//WHATWG//DTD HTML 4.01 plus Web Forms 2.0//EN"

     "-//WHATWG//DTD HTML 4.01 Transitional plus Web Forms 2.0//EN"

     "-//WHATWG//DTD XHTML 1.1 plus Web Forms 2.0//EN"

N.B.:

   * I think there should be a space between "HTML" and its version
     number.
   * I don't think there's any point extending HTML 4.0, that was
     superceded by HTML 4.01.
   * I believe it'd be more in keeping with other FPIs if we used "plus"
     rather than "+" or "with".

I'm not actually sure how much of a call there is for "-//WHATWG//DTD 
HTML 4.01 Transitional plus Web Forms 2.0//EN"; on the other hand what 
about DTDs for:

     "-//WHATWG//DTD XHTML Mobile 1.0 plus Web Forms 2.0//EN"

or:

     "-//WHATWG//DTD XHTML Basic 1.0 plus Web Forms 2.0//EN"

Regards,
