No, it doesn't, as is trivially shown. IE has _no_ support for XHTML, it
only has support for HTML4. If you give it XHTML, it just treats it as
HTML4, and every XHTML feature that it doesn't understand is treated as an
error which it then uses error-recovery to handle.

See http://www.hixie.ch/advocacy/xhtml for more details.
