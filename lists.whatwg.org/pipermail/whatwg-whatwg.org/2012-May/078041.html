> Good luck pushing Apple & Microsoft in implementing this. If we create this
> as a tag then we'd push every OS vendor to support it.

Mac OS X supports "Arabic", "Arabic - PC" and "Arabic - QWERTY" input methods. If none of these provide the functionality of "Arabizi" then please file a bug at <http://bugreport.apple.com/> and we can consider adding it for the whole OS. Mac OS X also supports third-party downloadable input methods. We are unlikely to support a browser-specific form of input methods.
