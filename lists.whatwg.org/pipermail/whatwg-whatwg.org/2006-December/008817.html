Hello,
While reading the spec today, I came across a possible typo in section 3.4.3:

"The xml:lang  attribute may only be used on elements of XML
documents. Authors must not use the lang attribute in HTML documents."

The second sentence should probably say "xml:lang", not "lang", since
HTML authors are actually supposed to use lang.
