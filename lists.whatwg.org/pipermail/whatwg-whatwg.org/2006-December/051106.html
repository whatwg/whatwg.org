Hi,

Many transcripts on the Web include timestamps, in particular chat logs. If 
chat Web applications are to use <dialog>, then those apply aswell. I'm sure 
there are other examples too.

I think <dialog>s should support a way to mark up timestamps. I'd suggest 
<time> in the <dt>:

   <dialog>
    <dt><time>00:01</time> zcorpan
    <dd>speaking of <dialog>... can <dt> include other stuff, like 
time stamps?
    <dt><time>00:01</time> Hixie
    <dd>zcorpan: not currently
   </dialog>
