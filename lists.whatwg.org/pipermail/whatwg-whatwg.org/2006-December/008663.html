Seeing how the <select> element is mostly used in the wild, I think an
'autosubmit' attribute on the <select> element will increase accessibility
because the current deployed solutions all depend on JavaScript, usually
without any kind of fallback to an ordinary submit button.

Seeing that having an onchange event on a <select> element triggering
submit on its form is such an oft-used solution, I think WF2 would provide
a good solution for future web forms and their accessibility by providing
an 'autosubmit' attribute that serves this purpose.

If such an attribute or mechanism have been discussed before, I'm sorry; I
don't read this list regularly -- it's much too crowded for that.
