On Wed, 6 Dec 2006, Alexey Feldgendler wrote:
> 
> http://whatwg.org/specs/web-apps/current-work/#elements lists the 
> elements which are considered void. The <command> and <event-source> 
> also have empty content model, so shouldn't they end up there as well?

They're there, but commented out. They're also missing from the parsing 
section. I have not yet added any of the new elements to the syntax 
section, so that (until HTML5 is in a more stable state) people can use 
the syntax section to write HTML4-level parsers without worrying about 
whether or not the new elements get accepted or not.
