On Dec 4, 2006, at 8:04 PM, Simon Pieters wrote:

> FWIW, HTML 3.0 had a <credit> element as child of <fig> for this  
> purpose.

Yes, though interestingly it was also for block quotations; see  
http://www.w3.org/MarkUp/html3/credits.html. The use of <credit> in  
<blockquote> seems to have been broader than the current use of  
<cite>, let alone the version in HTML 5; see http://www.w3.org/MarkUp/ 
html3/blockquotes.html. This model for a credit seems more applicable  
to the proposed <figure> -- I'd support adding that element to the spec.
