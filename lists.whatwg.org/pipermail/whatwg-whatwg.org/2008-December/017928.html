Are there any plans to add the width/height attributes for the input element to the HTML5 spec? It seems that all browsers (Opera, FF, Safari, IE) support width/height on <input type="image" in both quirks and standards mode. There are websites that depend on this behavior (the select-your-postal-code landing page at canadiantire.ca is one example) and it doesn't seem to be specced anywhere that I could find.
