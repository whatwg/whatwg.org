You can raise validator.nu bugs here:

  http://bugzilla.validator.nu/

You might want to raise a bug there with description "add support
for the sizes attribute on link" (or whatever).

I think the reason you get that error message is that the schema
validator.nu uses has not yet been updated to add the sizes
attribute to the link element.
