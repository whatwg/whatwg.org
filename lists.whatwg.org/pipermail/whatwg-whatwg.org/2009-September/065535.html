I'd like WHATWG to host a single document that contains HTML5, Web  
Workers, and everything that is in the source document but is being split  
out to separate specs.

I'd like this for two reasons:

1. The WHATWG copyright that was originally attached to the things that  
were split out doesn't seem to apply now when they are in separate specs  
with different copyright notices. http://svn.whatwg.org/webapps/source  
does not include the WHATWG copyright notice.

2. It's simpler to review Web Workers when the xrefs to HTML5 aren't  
broken. Even when gsnedders implements xdoc xrefs, it would still be  
annoying to jump back and forth between the two documents. (I guess the  
same applies to Web Database and the others, too.)
