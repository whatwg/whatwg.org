Certain DOM attributes such as the href DOM attribute[1] are implemented
differently than specified. The attribute is implemented as to return a
resolved link, not as equivalent to the href HTML attribute.

I also think that these kind of things should be more clearly defined in the Web
Forms 2 DOM interfaces[2] as the W3C DOM level 2 HTML[3] is not clear at all
about the relation between DOM attributes and HTML attributes such as action
for example.


[1]<http://whatwg.org/specs/web-apps/current-work/#href4>
[2]<http://whatwg.org/specs/web-forms/current-work/#dom>
[3]<http://www.w3.org/TR/DOM-Level-2-HTML/>
