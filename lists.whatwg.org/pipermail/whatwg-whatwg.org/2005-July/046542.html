>
> Certain DOM attributes such as the href DOM attribute[1] are implemented 
> differently than specified. The attribute is implemented as to return a 
> resolved link, not as equivalent to the href HTML attribute.

It's defined to reflect it, not as equivalent.

Reflect is defined carefully:
   http://whatwg.org/specs/web-apps/current-work/#reflect


> I also think that these kind of things should be more clearly defined in

to clarify existing features when they are poorly implemented.
