Two points:

   1. The 'scheme' attribute from HTML 4 is missing. If there's
      a reason for this, please include a note stating the reason
      for removal (and thereby make the removal explict). Note that
      certain metadata formats (e.g. Dublin Core) do seem to be
      using this attribute.
   2. Your recommendation not to give character encoding information
      inline is in direct conflict with the recommendations of the
      W3C I18N GEO WG:
http://www.w3.org/International/geo/html-tech/tech-character.html#ri20040215.101337371
      I hope that by the time this spec goes last call you are all
      in agreement as to what advice you want to give web authors
      wrt this issue.

~fantasai
