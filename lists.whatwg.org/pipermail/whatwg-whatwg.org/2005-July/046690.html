It should basically not happen at all. It appears that no browser has
implemented the 'may' from HTML4 and therefore we can now say browsers must not
trim.

Examples:
 checkbox:
  <input type="checkbox">
  <input type="Checkbox">
  <input type=checkbox>
  <input type=checkbox >
  <input type=
              checkbox >
 ...

 no checkbox:
  <input type="checkbox
                       ">
  <input type=" checkbox">
  <input type="checkbox ">
  ...
