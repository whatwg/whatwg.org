The proposal was purely for the sake of semantics. A <span class="lede"> styled appropiately via CSS works and looks fine. The element would obviously add no functionality. However, I though that this element would add more semantic richness and would be useful to news aggregators in particular as an alternative to using the first sentence (Google), the first paragraph (Yahoo) or the meta description(bbc).
