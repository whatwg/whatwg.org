<p class=MsoNormal>I’d like to propose that UAs should surface an bytes transferred
event when a form is being submitted. With so many large files being uploaded
using browsers today and every website that allows this having had to implement
their own upload progress indicator I think the Html 5 spec should support this
very common need.<o:p></o:p></p>

<p class=MsoNormal><o:p> </o:p></p>

<p class=MsoNormal>In Flash player, the FileReference object surfaces multiple
events such as<o:p></o:p></p>

<p class=MsoNormal>Select<o:p></o:p></p>

<p class=MsoNormal>Complete<o:p></o:p></p>

<p class=MsoNormal>IO/Error<o:p></o:p></p>

<p class=MsoNormal>Progress<o:p></o:p></p>

<p class=MsoNormal>Upload Complete - makes available the response from the web
server, which could be displayed to the end user<o:p></o:p></p>

<p class=MsoNormal><o:p> </o:p></p>

<p class=MsoNormal>The progress event provides bytesLoaded and bytesTotal and
with this information one can do the following:<o:p></o:p></p>

<p class=MsoListParagraph style='text-indent:-.25in;mso-list:l1 level1 lfo1'><![if !supportLists]><span
style='mso-list:Ignore'>1.<span style='font:7.0pt "Times New Roman"'>      
</span></span><![endif]>Show the user how much time before the file is uploaded
completely (in some cases it takes them 7-8 hours to upload afile)<o:p></o:p></p>

<p class=MsoListParagraph style='text-indent:-.25in;mso-list:l1 level1 lfo1'><![if !supportLists]><span
style='mso-list:Ignore'>2.<span style='font:7.0pt "Times New Roman"'>      
</span></span><![endif]>Using this information we can also calculate the rate
of transfer (upload speed).<o:p></o:p></p>

<p class=MsoNormal><o:p> </o:p></p>

<p class=MsoNormal>Ideally UA should:<o:p></o:p></p>

<p class=MsoListParagraph style='text-indent:-.25in;mso-list:l0 level1 lfo2'><![if !supportLists]><span
style='mso-list:Ignore'>1.<span style='font:7.0pt "Times New Roman"'>      
</span></span><![endif]>Show the upload progress along with the estimated time
left and upload speed. If this is provided at a control the site developers can
place this control and skin it they way they’d like to, or the statu bar
of the browser should show this information.<o:p></o:p></p>

<p class=MsoListParagraph style='text-indent:-.25in;mso-list:l0 level1 lfo2'><![if !supportLists]><span
style='mso-list:Ignore'>2.<span style='font:7.0pt "Times New Roman"'>      
</span></span><![endif]>Allow for filtering the types of files a user can
select by providing a semicolon delimited list of file extensions such as “*<b>.wmv;*.mp4;</b>*.mov”
etc.<o:p></o:p></p>

<p class=MsoNormal><o:p> </o:p></p>

<p class=MsoNormal>At a minimum, I think they should provide a Progress event
and UploadCompleted event.<o:p></o:p></p>
