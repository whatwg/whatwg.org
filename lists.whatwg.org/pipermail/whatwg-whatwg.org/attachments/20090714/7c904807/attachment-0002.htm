Target '_search' makes a link open in a sidebar (Opera) or sidebar-like
window (IE). For some offline web apps would be cool to open sidebar by
just one click. In other browsers (Firefox) web content could be open
in a sidebar only by creating a bookmark, marking it as to open in a
sidebar, use (click) that bookmark.<br>
<br>

all browsers but it is not taken as a standard. I would really like
this feature to be available in all browsers.<br>
