How about a hash attribute for all elements that link to external files (a, img, etc.)?<br><br>It would allow you to pass an MD5, SHA-1, SHA-256, or other hash to a user-agent for automatic comparison with the linked file.<br><br>I'd suggest a format where the hash algorithm is listed, followed by a forward slash and then the hash itself<br><br>Examples:<br>hash="MD5/9e107d9d372bb6826bd81d3542a419d6"<br>hash="SHA-1/2fd4e1c6 7a2d28fc ed849ee1 bb76e739 1b93eb12"<br>hash="SHA-256/d7a8fbb3 07d78094 69ca9abc b0082e4f 8d5651e4 6d3cdb76 2d02d0bf 37c9e592"<br><br>See my thread over at mozillaZine for more info and other ideas: http://forums.mozillazine.org/viewtopic.php?p=2234056<br><br>--Wulf<br><p>
