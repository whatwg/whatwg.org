<p class=MsoNormal><font size=2 color=navy face=Arial><span lang=EN-US
style='font-size:10.0pt;font-family:Arial;color:navy'>Uniqueness of tokens can
be determined in O(n) only* if the tokens are ordered in the source (any order
would do) but there is no such requirement, and it cannot be required for
compatibility with the content in the wild and because the standard supports
inserting new tokens.<o:p></o:p></span></font></p>

<p class=MsoNormal><font size=2 color=navy face=Arial><span lang=EN-US

this issue and proceed as if the tokens were ordered. &nbsp;The result would be
that remove would fail, or it would run in quadratic time.<o:p></o:p></span></font></p>
