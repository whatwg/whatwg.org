Should media elements that are loading, haven't reached HAVE_CURRENT_DATA, but are not actually in the document delay the load event for their owner document? The spec doesn't say elements that delay the load event have to be in the document, but it also doesn't say specifically which document's load event an element delays.<br>
<br>Out-of-document image loads can delay the load event for their owner document, so I think it makes most sense to allow it for media elements too.<br><br>Context: <a href="https://bugzilla.mozilla.org/show_bug.cgi?id=479711">https://bugzilla.mozilla.org/show_bug.cgi?id=479711</a><br clear="all">
