On 2/20/12 9:08 PM, Rodger Combs wrote:
> Mac OSX 10.8 (Mountain Lion), along with most mobile operating systems, has a system-wide sharing functionality. Could the Web Intents API add a "SHOULD" or "MAY" clause for adding web intents to the system-wide "share sheet" or similar functionality, rather than just on the web? It fits with a few other parts of the HTML5 spec that move towards OS integration (e.g. notifications, which would fit well with Android/iOS/OSX10.8's built-in notification centers).

I certainly see this happening with Chrome OS in place of the file 
system handler they currently have.

I'd imagine it'll make its way to the Windows shell in the future, via 
Chrome: think of the "Send to" context menu amongst others. At some 
point, it will be very easy.
