In WF2 I can prefill a select list with external data pointed to by a 
data attribute like so:

<form>
  <label>Choose the species:
     <select name="species" data='examples/species.xhtml'></select>
   </label>
  <label><input type="submit" value="Send data" /></label></p>
</form>

However it strikes me as funny that I can't prefill a repeating form. e.g.

<div id="observation" repeat="template">
   <label>Species:
     <input type="text" name="species[observation].name"
            data="http://www.example.com/values.html"/>
   </label>
</div>

At least I don't think I can. This seems so obviously useful I can well 
believe I might have overlooked the relevant section of the spec. if so, 
could someone please point me to it? If not, would anyone else like 
something like this?
