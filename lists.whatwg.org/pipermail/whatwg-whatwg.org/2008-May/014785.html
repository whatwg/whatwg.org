Is there any reasonable chance an element for a dialog box might end up being added to XForms?  (There is a proposal mentioned on the XForms wiki [1] for a possible dialog element for XForms 2.0, but I have no idea how much of a chance that proposal has as opposed to extending the message element.) I know that XHTML 5 + XForms isn't a major concern, but I do think that avoiding a problem for those that will be using various flavors of XHTML with  XForms is worth addressing now.  If XForms were to add an explicit dialog box element, what name other than <dialog/> would be appropriate?
