The CSS WG has published an updated Working Draft of the
CSS Cascading and Inheritance Module Level 3:

     http://www.w3.org/TR/css3-cascade/

Changes since the previous WD include the addition of scoped rules
   http://www.w3.org/TR/css3-cascade/#cascade
the 'default' keyword
   http://www.w3.org/TR/css3-cascade/#default
and the 'all' shorthand
   http://www.w3.org/TR/css3-cascade/#all
and the whole draft has been substantially revised and updated against
CSS2.1.

Major changes are listed at:
   http://www.w3.org/TR/2013/WD-css3-cascade-20130103/#changes

We'd particularly appreciate a review of Chapter 4, Cascading and Inheritance.
   http://www.w3.org/TR/css3-cascade/#cascade-and-inheritance

Please send any comments to this mailing list, <www-style at w3.org>, and
please, prefix the subject line with

     [css3-cascade]

(as I did on this message).

For the CSS WG,
