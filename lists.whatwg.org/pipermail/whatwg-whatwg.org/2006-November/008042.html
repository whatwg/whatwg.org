On Tue, 2006-11-28 at 15:56 -0500, Jeff Seager wrote:

> I think the artist's name and copyright notice are worthy "advisory
> information" to be included in the TITLE attribute of <img>, <embed>, or
> <object>, and this has been possible for some time.

We're dealing with a hypertext medium. Attributes should either be a URI
(like the "cite" attribute) or just plain text (like the "title"
attribute"). As attributions often need to be either plain text, or
hyperlinks, or both, they should not be entirely relegated to an an
attribute. I like the "cite" attribute for quotations because it allows
you to jump straight to the source. It doesn't sound like it would have
the same utility for figures.
