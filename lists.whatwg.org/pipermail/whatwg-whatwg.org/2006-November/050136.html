I finally got around making the following simple tool:

    http://html5.org/tools/specification-diff

It's a small Python script that lets you generate unix diffs of Web
Applications 1.0. There's a form where you can enter two numbers. An
example would be:

    http://html5.org/tools/specification-diff?from=275&to=276

I hope this can help everyone who's reviewing the document now and then to
see what has been changed. So you don't have to reread the document
everytime or run your command line tool to generate a diff.

It may take some time to generate a diff so you have to be patient. If you
have any questions on usage or encounter some problems feel free to
contact me offlist.
