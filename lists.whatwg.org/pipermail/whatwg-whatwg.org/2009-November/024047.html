How would HTML 4.01 clients respond to multiple BASE tags in a HTML 5
document?  Even with extra attributes to say "Restrict this to a particular
content-type," you'd probably see some less-than-desirable side effects -
HTML 4 clients wouldn't care about them.

The BASE tag affects how the document's other resources are loaded.  Older
clients reading the document would probably break.
