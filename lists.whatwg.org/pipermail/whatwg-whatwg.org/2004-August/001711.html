>    I don't understand why you wouldn't be able to style an HTML element. 
> True, you can't necessarily style the widget directly, since 
> implementation of an HTML element is largely left to vendors, but you 
> could certainly apply CSS properties to it, and perhaps even CSS

If we are talking about 'xhtml:html' than I can assure you that *that* 
element is certainly styleable in the better browsers.

The only special thing is the 'background' property, which gets applied 
to the canvas and not to the HTML element itself.

(This is correctly implemented in at least the latest versions of Opera 
and Mozilla.)
