For a poem whitespace can be quite important. I think PRE would be much 
better for poems in general.

About the examples, why not use some Lorum Ipsum <http://lipsum.com/>

Also, abusing P for grouping form controls seems just odd. That does not 
add any semantics. Although it is quite commonly used to do that, I 
don't think HTML5 should advocate such usage. And if we want it, it 
should be explicitly stated in the specification. (It would be as 
confusing as DL is now. Specifying that you use it for definitions and 
giving an example of people having a conversation.)
