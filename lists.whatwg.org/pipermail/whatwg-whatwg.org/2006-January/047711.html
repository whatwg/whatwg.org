Some authors set IDs on every 'section' of their pages - on heading levels
for instance - to make it a bit easier for others to hyperlink directly to
sections. I don't think a HTML spec should require this, but with the trend
towards using automated authoring systems, this could get in more common use
as such systems could easily automatically insert IDs for heading levels.

But XPointer looks like it might be more realistic. I wasn't aware of it
until just now. Is it being implemented in any user-agents already?
