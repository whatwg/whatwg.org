     I'm using the appcache since it's adoption on major browsers and i 
think it lacks of a section to specify fallbacks with a specific timeout

the main issue still remaining in order to have the same experience than 
installed apps.

     It could take the form of a section like this (with a 2 seconds 
timeout) :
TIMEOUT 2:
onlineres.html fallbackres.html

     Or maybe be done with the connection types :
BANDWIDTH EDGE:
onlineres.html fallbackres.html
     ...

     Is it planned to have some similar mechanism ?
