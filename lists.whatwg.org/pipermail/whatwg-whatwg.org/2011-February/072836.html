Implementation of which?

The problem with prefetching immediately on src set is that you have no 
idea when or whether the node will get inserted.  So you have to keep 
the data alive... for how long?  From a user's point of view, that's a 
memory leak pure and simple, if the node never gets inserted into the DOM.
