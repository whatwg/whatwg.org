Consider the following testcase:

<!DOCTYPE html>
<a href="http://www.example.com" onclick="return 0">Click me</a>

Should clicking the link load www.example.com?

That is, does the return value of the onclick handler cancel the event if it's 
== to false, or only if it's === to false?

In shipping Geckos it only cancels if the return value === false; I suspect 
other browsers behave similarly but it could use testing.
