Hi,

I tested "&apos;" in text/html[1][2]. The following browsers treat it as if 
it were "&#39;":

* Mozilla
* Opera
* Safari
* Lynx
* Konqueror

The following browsers treat it as if it were "&amp;apos;":

* Windows Internet Explorer 4.0 - 6.0
* Macintosh Internet Explorer 5.2
* Netscape Navigator 4.78

Should "&apos;" be a valid charater reference in text/html? If not, what 
would be correct error handling?
