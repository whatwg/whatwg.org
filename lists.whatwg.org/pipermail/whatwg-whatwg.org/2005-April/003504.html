A submit button may be associated with multiple forms through the form 
attribute. Theoretically, this should allow the button to submit several 
forms at the same time when pressed. This could get really messy, so I 
propose that a submit button should only be allowed to be associated 
with one form at a time. (Along the same line, hitting enter when focus 
is in a text field should only submit one form.)
