I'm disappointed you got that impression, that's actually the opposite
of what I was trying to get across, as soon as I read it back after
posting I was disappointed with how it sounded.  The non-IE user
agents represented here are all really good user agents, however in
the specific situation of extensibility with just scripting, they're
pretty weak (with the exception of mozilla of course) and rather than
giving us some elements that we can go away and implement using the
extensibility mechanisms of IE.  I'd much rather see the effort spent
purely in developing extensibility mechanisms in all the user agents,
then we could achieve the same as Web Forms 2.0, and more.

I'm disappointed that the developers of 3 very good browsers are
wasting their time tweaking things around the edges of things that
scripters can already do, when there are already many interesting
technologies out there they could be implementing instead.
