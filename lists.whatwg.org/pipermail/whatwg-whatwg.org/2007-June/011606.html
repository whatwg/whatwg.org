Hi all,

the Annodex Association and Xiph Foundation are in the process of
implementing a small-footprint library and firefox plugin called
OggPlay that provides native Ogg Theora support.

There's a javascript API in the process of being specified - and I
think it would be valuable for others to have a look at and provide
input. It may also be possible to include such a javascript API into
the HTML5 spec for the <video> element.

Check it out here: http://wiki.xiph.org/index.php/OggPlayJavascriptAPI

Feedback should probably go in copy to the theora at lists.xiph.org mailing list.

Regards,
Silvia.
