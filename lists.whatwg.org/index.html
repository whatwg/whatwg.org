The WHATWG mailing lists are no longer active.

Discussions now happen on the repositories for the individual standards, and whatwg/meta can be used for issues without a logical home.

Archives

A static archive of the whatwg@whatwg.org mailing list 2004-2019 is available, but has a gap in 2014-2017. A searchable and complete archive is also maintained by the W3C.

Static and complete archives of the other lists are also available:

  • commit-watchers@lists.whatwg.org: commits to HTML5's Subversion repository
  • help@lists.whatwg.org: the place for HTML5 authors to help other HTML5 authors
  • implementors@lists.whatwg.org: WHATWG specification implementors hangout

If you've found a broken link to these archives, please file an issue. We may be able to recover it based on context.
